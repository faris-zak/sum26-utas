# Smart Student Guide Event Website

Arabic-first bilingual event demonstration for **دليل الطالب الذكي / Smart Student Guide**. The experience is built with plain HTML, CSS, and JavaScript and does not call an AI provider.

## Run locally

```powershell
npm.cmd run serve
```

Open `http://localhost:4173`. The first visit caches the four core destinations for offline use.

## Verify

```powershell
npm.cmd test
```

All university policies, forms, dates, and metrics in this project are fictional event-demo content. Metrics are explicitly presented as projected outcomes.
